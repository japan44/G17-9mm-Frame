# G17-9mm-Frame
 This 9mm Glock 17 3rd gen frame uses steel rails which can be fabricated with the templates provided. 16 gauge steel
 should be used. It requires a 3 pin lower parts kit with a 2 pin locking block. The smaller pin for the locking block
 is used to secure the front rail. Any 3rd gen g17 slide and barrel can be used.
